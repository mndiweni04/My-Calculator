My Calculator
================================================================================

This project contains the code to a basic calculator(no scientific functions).
The code has comments that explain the different functions and methods that
have been used in this program.


To open the main form in the GUI Builder:

1) In the IDE's Projects window, navigate to the src package node within the 
Source Packages directory.

2) Double-click the node of the CalcFrm.

The IDE launches the GUI Builder and displays the calculator.
